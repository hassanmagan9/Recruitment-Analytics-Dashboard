# Recruitment Analytics Dashboard

## Project Overview
The Recruitment Analytics Dashboard is an interactive data analytics project designed to provide insights into the recruitment and hiring process.

This dashboard allows HR teams and recruiters to monitor hiring performance, track candidate progression through recruitment stages, and evaluate the effectiveness of different recruitment channels.

## Dashboard Features
The dashboard provides a comprehensive view of the recruitment pipeline including:

- Open Positions Tracking – Displays the number of available positions across different departments.
- Application Analysis – Tracks total applications and identifies the main recruitment channels.
- Recruitment Pipeline – Visualizes candidate progression through stages such as screening, interviewing, evaluation, offering, and confirmation.
- Hiring Performance Metrics – Includes key metrics such as total hires, average days to hire, and rejection rates.
- Recruitment Trends – Monthly recruitment activity and hiring performance analysis.
- Candidate Insights – Provides details about applicants and job positions.

## Key Insights
This dashboard helps organizations:

- Understand the efficiency of the recruitment process
- Identify bottlenecks in the hiring pipeline
- Evaluate the effectiveness of recruitment sources
- Monitor hiring trends over time
- Improve talent acquisition strategies

## Tools & Technologies
- Microsoft Power BI
- Data Modeling
- Data Visualization
- Recruitment Data Analysis

## Project Objective
The goal of this project is to demonstrate how data analytics can improve recruitment decision-making and optimize the hiring process.
